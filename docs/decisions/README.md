## Writing an ADR

Use this template:

```markdown
# ADR-NNN: [Title]

Date: YYYY-MM-DD
Status: Proposed | Accepted | Superseded

## Context
What problem or situation led to this decision?

## Decision
What did we decide to do?

## Consequences
What are the trade-offs or follow-ups?
```

Number sequentially: 001, 002, 003, etc.
