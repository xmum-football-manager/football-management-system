ABANDONED CONFLICTING CHANGES — from branch `finished-latest-tests` (commit 77ec7b2)
================================================================================

Decision: `main` is the source of truth. The big commit 77ec7b2 ("Finished latest
tests and changes", 129 files) was NOT merged because 6 of its files conflict with
main's lib/format refactor (commits db436c5 + ec5b610) and the knockout docs.

main was left untouched at commit 0a24afd. The 125 non-conflicting files in 77ec7b2
were NOT pulled in either, because the new screens they add depend on the rewritten
versions of the components below — you cannot take half the rewrite.

This file records WHAT those 6 conflicted files did, so the work can be re-applied
later on top of main if wanted. The full code still lives on branch
`finished-latest-tests` (and origin/finished-latest-tests on GitHub).

To see any file's full version:
    git show finished-latest-tests:<path>

--------------------------------------------------------------------------------
1. web/components/HeroLive.tsx   (biggest change — full rewrite)
--------------------------------------------------------------------------------
Features added in 77ec7b2 that are NOT in main:
  - Live match clock now ticks every 1 SECOND and shows MM:SS (main: every 30s,
    minutes only). New <LiveClock> with a period indicator ("1H" / "2H") derived
    from a new `minutesPerHalf` prop. Uses suppressHydrationWarning for SSR drift.
  - New props: `match` (required), `allMatches?`, `metaText?`, `minutesPerHalf?`.
    (main's HeroLive uses `variant: 'live'|'nextup'|'done'` + optional `match`.)
  - Team recent-form strip: teamForm() computes last-5 W/D/L from allMatches;
    rendered as colored pips via <FormStrip>.
  - Team crests: real uploaded logo via mediaUrl(team.logo_path), falling back to
    teamColor()/teamCode() from lib/team-style (instead of teamInitials()).
  - Animated <ScoreDigit> (tick-in animation on score change) and a <BallIcon> SVG.
  NOTE: This is the component the SCOREKEEPER-style MM:SS running timer request maps
  to. It already existed in 77ec7b2 but is abandoned with this decision.

================================================================================
COMMON THREAD
================================================================================
Almost every conflict is the same theme: 77ec7b2 introduced an IMAGE/UPLOAD system
(lib/storage.ts mediaUrl(), lib/team-style.ts teamColor/teamCode, ImageUpload,
the 20260607092314_add_entity_images.sql migration) and replaced the text-initials
avatars everywhere with real uploaded logos/photos. main instead consolidated the
old initials logic into lib/format.ts (teamInitials). The two approaches edit the
same lines, hence the conflicts.

If you later want the uploads feature + the 1-second MM:SS match clock on main,
cherry-pick 77ec7b2 onto main and resolve these 6 files by taking the 77ec7b2 side
(it has to win — the new screens depend on it), keeping main's lib/format only where
it doesn't clash.
