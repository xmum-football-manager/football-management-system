# Contributing to Football Tournament System

Thanks for contributing! Here's how to safely add code.

## Workflow

1. **Create a branch** from `main` (or `development` if specified)

   ```bash
   git checkout main
   git pull origin main
   git checkout -b feature/your-feature-name

2. Make your changes (code, docs, etc)
3. Commit and push
   ```bash
   git add .
   git commit -m "feat: add new feature"
   git push origin feature/your-feature-name

4. Open a pull request (PR) on Github
   - Go to the repository → "Pull requests" → "New pull request"
   - Set base = main, compare = your-branch-name
   - Add a title and description
   - Click "Create pull request"

5. Request a review (tag your team lead)

6. After approval, merge the PR (or let the lead merge it)
